## Wichtige html - Tags

-   `<html>` -> `<head>` -> `<body>`
-   `<title>` (im head)
-   Paragraph `<p>`
-   Bilder `<picture` `<img>`
-   Links `<a href="..."`
-   Definitionen `<dl> .. <dt> .. <dd>`
-   Abschnitt (Division) `<div>`
-   überschriften `<h1> ... <h6>`
-   Listen: `<ul> <li>` (unordered)
-   Liste geordnet: `<ol> ... <li>`
-   Knopf `<button>`
-   Eingabe: `<input>`
-   Tabellen: `<table>`
